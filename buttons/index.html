Buttons

The S2 approach of using a generic .button class combined with another class for specific color treatment is being deprecated in favor of the single class approach.

Deprecated Classes

All instances of the following classes will be replaced.

  • .button
  • .button .primary
  • .button .secondary
  • .button .action
  • .button .google-btn
  • .button .twitterblue
  • .button @button-gradient-sky-blue()
  • .button @button-gradient-orange()
  • .button .cta span.big
  • .button .subtle
  • .button .primary .wide
  • .button .sooth
  • .button .primary (on dark)
